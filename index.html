    What is the "The Web"?

    The Web is everything.

    If we consider:

    "the Web" as the "Internet Service",
    "Websites" as the "Interface"
    and "Mobile" as the "Environment"

    In order to serve the same "Websites"/"Interface"
    to different "Mobile"/"Environment" we need a technology to be smart enough to adapt to the new environment

    "Websites"/"Interface" will be just one of many interfaces we will use to interact with the "the Web"/"Internet Service"

    Developing across multiple platforms and browsers

    Responsive Web Design

    Responsive Web Design creates a "Unique" and "future-friendly" Web capable of flowing through all platforms, devices and brands

    The 123 of RWD

    • 1- A flexible, grid-based layout
    • 2- Flexible images and media
    • 3- media queries

    What is a future-friendly Web

    Focus on Content:

    Keep it simple. Less is more.
    Analyze our content and establish strong content strategies to ensure our content is meaningful to our users and our businesses.

    Content/Mobile First:

    Based on the minimum requirement building up a core experience.

    Fluid Content:

    Content needs to be fluid, like water. When water flows into a larger or smaller part of a river it instantly adapts to the shape and size of it's environment (Master Lee)

    Progressive Enhancement:

    Apply progressive enhancement to things as basic as screen size, bandwidth, JavaScript support, and performance. building up a core experience

    Device capabilities detection:
